Selfhosted Projects

What's it about?

  • not about virtual build machines
  • not about Docker
  • about a project setup thats maintainable by all developers

requirements for a software project

  • every developer in the team can produce artefacts (CI, Staging, Production)
  • build system executes on every dev machine
  • Jenkins/Travis executes debugable/reproduceable build code
  • The build system is part of the project
  • Jenkins/Travis can build past and feature versions without config change

problems we might face with these requirements

  • dependencies [solved]
  • jenkins setup
  • credencials

Meet Lanyard

Small helper gem for the keychain handling https://github.com/wooga/lanyard

contians

  • cli tool
  • ruby module with helper functions
  • simple security tool wrapper
  • rake task creator

cli


Usage:
  lanyard use-keychain [options] (<keychain> <password>)...
  lanyard unuse-keychain <keychain>...
  lanyard (-h | --help)

rake simple


require 'rake/lanyard_task'

lanyard = Rake::LanyardTask.new

rake enhanced


require 'rake/lanyard_task'

lanyard = Rake::LanyardTask.new { |t|
    t.namespace_name = :keychain
    t.keychain_path = "path/to/keychain-copy.keychain"
    t.keychain_password = "keychain password"
    t.copy_keychain_from "path/to/keychain.keychain"
}
